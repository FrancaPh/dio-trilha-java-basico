## Descrição
Esse repositório foi criada para postar as resoluções dos desafios dos labs presentes no bootcamp da **[DIO](https://web.dio.me)** com parceria do Santander, [Santander 2024 - Backend com Java](https://web.dio.me/track/santander-2024-backend-com-java). Cada um dos branches representam um desafio diferente e possuem sua descrição, proposta do desafio e a resolução.
